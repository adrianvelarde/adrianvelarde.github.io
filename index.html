C# - Turing Machine

This assignment demonstrated knowledge of the way a Turing Machine operates and the concept behind the virtual tape. All the textboxes that are on the form are appended dynamically when the user selects an amount of input, rules, accepting states, and when they generate the grid. The tape that is created displays the word that is typed and is accepted or rejected depending on the language's transition rules. The language in the example only includes words that have an odd number of a's and odd number of b's. The 'x' in the input word makes it rejected by the language and the virtual tape highlights the letter that made the word rejected. The user can also move the tape side to side to view each letter and the current letter is highlighted gray.

Java - Secret Message

This assignment was given solely for the purpose of 2D-array manipulation. The encryption key is the number given to each column in the 2D-array. The message, here it was "I want an A in this class", is then encrypted by reordering the 2D array so that the key is in numerical order. Decryption occurs through the reversal of the keys and columns into their original positions.
